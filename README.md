# Eger Solutions Landing Page

Site estático desenvolvido para a assistência técnica **Eger Solutions** em Caçador/SC.

## Como publicar no GitHub Pages

1. **Criar Repositório:** Crie um novo repositório no GitHub (ex: `eger-solutions-site`).
2. **Subir Arquivos:** Faça upload dos arquivos `index.html`, `styles.css`, `script.js` e a pasta `assets/` (contendo sua logo).
3. **Ativar Pages:**
   - Vá em **Settings** (Configurações) do repositório.
   - Clique em **Pages** na barra lateral esquerda.
   - Em "Build and deployment", escolha a branch `main` e a pasta `/ (root)`.
   - Clique em **Save**.
4. **Acessar:** Em poucos minutos, seu site estará disponível em `https://seu-usuario.github.io/nome-do-repositorio/`.

## Personalização
- **Logo:** Substitua o arquivo em `assets/logo.png` pela sua logo original.
- **Cores:** Caso queira ajustar os tons, altere as variáveis no topo do arquivo `styles.css` dentro de `:root`.
